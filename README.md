This repo contains the Jupyter notebook used to generate figures for this paper:

> O’Sullivan D, L Bergmann. 2026. Using MapWeaver to Make Tiled and Woven Maps of Multivariate Thematic Data. _Cartographic Perspectives_ **108**. doi: [10.14714/CP108.2109](https://dx.doi.org/10.14714/CP108.2109)

Note that figures were made with 0.0.7.19 version of `weavingspace` installed via `pip install weavingspace`.
